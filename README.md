# svg-to-hiccup

> Convert your svg icons to hiccup.

Passes them through [svgo][svgo] to optimize and minify them and spits out valid hiccup.


## Usage

``` sh
$ npx nbb -m svg-to-hiccup test.svg
[:svg {:xmlns http://www.w3.org/2000/svg, ....
```
